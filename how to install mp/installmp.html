STEP 1
Download the tools (listed above)

STEP 2
Verify the files of Pizza Tower

STEP 3
Open Xdelta UI, and set the patch file as the .xdelta file from the multiplayer mod (it's inside the zip)

STEP 4
In Xdelta UI, set the source file to data.win inside your Pizza Tower install folder

STEP 5
Set the output file as data.win, and save it SOMEWHERE BESIDES THE PIZZA TOWER INSTALL FOLDER

STEP 6
Click patch

STEP 7
Drag and drop the new data.win into the pizza tower install folder, and replace file when it asks

STEP 8
If you plan on using a controller, set the controls with steam input (it's a community layout, make sure you have show all bindings on)

STEP 9
If you don't have anyone next to you, use Remote Play Whatever to invite your friends to pizza tower

STEP 10
Make sure your friend's controls are set as well

STEP 11
Success
